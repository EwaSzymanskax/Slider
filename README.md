# Slider
> Slider with autoplay. 
> Live demo: https://ewaszymanskax.github.io/Slider/

## Technologies Used
- React.JS
- CSS

## Setup
In the project directory, you can run:

npm start

Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
